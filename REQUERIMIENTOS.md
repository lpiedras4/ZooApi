# Requerimientos de la API

## Descripción General

Esta API REST está diseñada para gestionar la información de un zoológico. Permitirá administrar los hábitats disponibles y los animales que residen en ellos, facilitando la organización y asignación de animales a sus respectivos espacios.

El sistema proporcionará endpoints para la creación de entidades y la gestión de las relaciones entre animales y hábitats.

## Entidades

### 1. Habitat
Entidad que representa los hábitats del zoológico donde residirán los animales.

### 2. Animal
Entidad que representa los animales del zoológico.

## Funcionalidades Requeridas

1. **Crear Habitat**: Capacidad para crear nuevos hábitats en el sistema.

2. **Crear Animal**: Capacidad para crear nuevos animales en el sistema.

3. **Asignar Animales a Habitats**: Capacidad para asignar uno o más animales a un hábitat específico.
